# Sistema de Auditoria Documental e Validação Socioeconômica

### Projeto de Conclusão de Curso - Sistemas de Informação (FHO)

Este sistema automatiza a triagem de documentos socioeconômicos (holerites, comprovantes e RGs) utilizando Visão Computacional para otimizar processos de bolsas de estudo.

## Tecnologias Utilizadas
- **Frontend:** React (Vite) + Tailwind CSS
- **Backend/OCR:** Python (FastAPI) + OpenCV + Tesseract
- **Banco de Dados:** MySQL
- **Modelagem:** Astah UML

## Estrutura do Repositório
- `/front-bolsas`: Interface React para alunos e auditores.
- `/motor-ocr`: Scripts Python de processamento de imagem e extração.
- `/docs`: Documentação, diagramas e atas de reuniões.

## Funcionalidades Planejadas
- [ ] Upload de documentos (RG, CPF, Holerites).
- [ ] Cadastro de composição familiar e renda per capita.
- [ ] Extração automatizada de campos financeiros via OCR.
- [ ] Painel de conferência com indicação de confiança da IA.
- [ ] Auditoria humana com correção manual de dados.
