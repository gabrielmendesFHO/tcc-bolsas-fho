# tcc-bolsas-fho
Projeto de Conclusão de Curso (2026) desenvolvido para o Centro Universitário Hermínio Ometto (FHO). O sistema utiliza Visão Computacional para automatizar a triagem de documentos em processos de bolsas de estudo, otimizando a análise de renda per capita e reduzindo a carga operacional das secretarias acadêmicas.
